# 🤝 Contribuer à DiceForge

Merci de l'intérêt que tu portes au projet ! Voici comment contribuer.

---

## 🚀 Par où commencer ?

1. **Fork** le repo
2. **Clone** ton fork localement
3. Crée une branche pour ta contribution
4. Fais tes modifications
5. Ouvre une **Pull Request**

```bash
git clone https://github.com/TON_PSEUDO/diceforge.git
cd diceforge
git checkout -b feature/ma-fonctionnalite
# ... tes modifications ...
git commit -m "feat: description de ta contribution"
git push origin feature/ma-fonctionnalite
```

## 📋 Conventions

### Messages de commit
```
feat: nouvelle fonctionnalité
fix: correction de bug
docs: documentation
style: mise en forme
refactor: refactoring
```

### Code
- Commente en français ou en anglais
- Une fonctionnalité = une Pull Request
- Teste avant de soumettre

---

## 🐛 Signaler un bug

Ouvre une **Issue** avec :
- Description du problème
- Étapes pour reproduire
- Ton OS et version de Node.js
- Capture d'écran si possible
